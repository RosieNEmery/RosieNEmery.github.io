QWI

Personal Project + Comission
Date : 2024
Qwi is a speculative cave dweller, comissioned as part of studio Transmoderna's upcoming installation ECHO. Qwi started out as a personal project inspired by sea slugs.

The project was completed using Houdini and Nuke, rendering done in Redshift.
