MARICI

Personal Project
Date : 2023
Marici is a fully digital headpiece, tracked and comped onto live action footage. The project centres around a headpiece influenced by my interest in fashion and cultural heritage.

Throught this project I got to practice and implement techniques for live action integration. I learnt a lot from the process and have now come to better understand what is needed from a live action shoot to produce the best output.

I was responsible for the headpiece design and all CGI elements, comp, cut and sound. The project was completed using Houdini, Blender, Nuke and Resolve with rendering done in Redshift.

Render of the final design; the design is inspired by south asian bridal headpieces and the S/S21 Iris Van Herpen collection.

(Above) 3D scans taken of the environment and head model to help with intergration. (Bottom right) Vellum simulation of the chains, using packed spheres for collisions. (Bottom left) Animation of a single armature which was then instanced and offset for the final design.

Rough HDRI created from chrome ball image exposures.
