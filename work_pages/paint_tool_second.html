PAINT TOOL + BRUSH

Personal Project
Date : 2020
Starting from my previous paint tool setup, I added a procedural paint brush texture to achieve a more visually rich look.

The brush texture is created by copying lines with varying thickness and opacity to the tool's underlying painting trails.

An example of the starting base curves (top right), generated brush lines (bottom right) and final render with varied line width and opacity.

Paint tool applied to a 3D scan (scan done with a photogrammetry app on an ipad).
